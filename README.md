# Validador-TISS-de-xml
Validador TISS de xml
Validar arquivo padrão TISS via Site do validador TISS

Sobre este fluxo de automação:
Esse fluxo permite a sua empresa da área de Saúde fazer uso do site https://www.validadortiss.com.br/.
Lembrando que você deve criar um usuário e senha para poder usar o envio de arquivos XML do padrão TISS para ser validade pelo site.
